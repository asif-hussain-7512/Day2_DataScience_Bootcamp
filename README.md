# 📊 Northwind SQL & Pandas Analysis

## 📌 Project Overview

This project is a part of the **Epochs '26 – Assignment 2**. The objective is to analyze the Northwind SQLite database using SQL to answer real-world business questions and use Pandas for exploratory data analysis.

The project demonstrates SQL querying, data extraction, and business insight generation from transactional sales data.

## 🎯 Business Questions

This project answers the following business questions:

1. What are the Top 10 Selling Products?
2. Who are the Top 10 Customers by Revenue?
3. What are the Monthly Sales Trends?
4. Which Product Categories Generate the Highest Revenue?
5. How Frequently Do Customers Place Orders?

---

## 📈 Exploratory Data Analysis

The SQL query results were imported into Pandas for analysis and visualization.

The notebook includes:

- Data loading from SQLite
- SQL query execution
- DataFrames for each business question
- Charts for key metrics
- Summary statistics
- Business insights

---

## 📊 Business Insights

- A small number of products account for a significant share of total sales.
- Revenue is concentrated among the top customers.
- Monthly sales exhibit noticeable fluctuations over time.
- Some product categories consistently outperform others in revenue.
- Customer purchase frequency varies, indicating opportunities for customer segmentation and loyalty programs.
